Welcome to Sticky Notes!

This is a CSS Grid demonstration showing how to create a masonry-style layout of sticky notes.

CSS Grid

This project uses CSS Grid for layout without any flexbox or positioning.

The notes automatically flow into available grid spaces.

Different Sizes

Notes can have different heights based on content.

The grid automatically adjusts to accommodate various sized notes.

This creates a dynamic, visually interesting layout.

Try resizing your browser window to see how the grid responds!

CSS Variables

This project uses CSS variables for colors, making it easy to maintain and update.

Responsive Design

The grid adjusts columns based on screen size.
