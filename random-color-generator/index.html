How It Works

This app generates random colors and displays them in different formats:

  • HEX: #RRGGBB format (e.g., #FF5733)
  • RGB: rgb(red, green, blue) with values from 0-255
  • HSL: hsl(hue, saturation%, lightness%)

Click on any color in the history to view it again.
