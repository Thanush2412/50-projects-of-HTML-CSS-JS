About This Game

This spelling game uses the Web Speech API for text-to-speech pronunciation and introduces JavaScript Generators for word sequence management.

How to Play:

  1. Select a difficulty level
  2. Click "Start Game" to begin
  3. Listen to the word pronunciation
  4. Type the correct spelling using the keyboard
  5. Use hints if you get stuck (limited number available)
  6. Complete as many words as you can!

New JavaScript Concepts:

  • Generators & Iterators: Managing word sequences efficiently
  • Speech Synthesis: Pronouncing words for the player
  • Destructuring: Simplified variable assignment
