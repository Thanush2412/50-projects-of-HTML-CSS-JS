What is CSS Grid?

CSS Grid Layout is a two-dimensional layout system for the web. It lets you organize content into rows and columns.

What property creates a grid container?

display: grid;

How do you define columns in CSS Grid?

grid-template-columns: 1fr 1fr 1fr;

This creates three equal columns.

What is the fr unit in CSS Grid?

The fr unit represents a fraction of the available space in the grid container.

What is a grid gap?

Grid gap (or gap) defines the space between grid items. It's a shorthand for row-gap and column-gap.

What is grid-auto-rows?

grid-auto-rows specifies the height of automatically generated grid rows.
